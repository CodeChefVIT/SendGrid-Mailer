## Instructions to run
```
$ git clone https://github.com/CodeChefVIT/SendGrid-Mailer
$ cd SendGrid-Mailer
$ npm i
$ Ask for env credentials
$ Add an array of emails to the mailingList
$ npm run dev
$ Open postman and use localhost:3000/sendMails
```
